#### All hashes and ctime with option "Detailed"
  * `./chk_dpl.py -d 1 /full/path/folder/fileX`
  * `./chk_dpl.py --detailed 1 /full/path/folder/fileX`

output: `md5 hash | ctime | path`

Alternative output: `None | 0 | full/path/folder/fileX`
